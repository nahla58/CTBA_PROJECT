💡 HOW EMAILS ARE SENT (QUICK OVERVIEW)

  1. CREATE BULLETIN (title, body, regions, CVEs)
           │
           ▼
  2. QUEUE FOR DELIVERY (async processing)
           │
           ▼
  3. PROCESS QUEUE (background thread)
           ├─ For each region:
           │  ├─ Get recipients
           │  ├─ Render HTML
           │  ├─ Send via SMTP
           │  └─ Log delivery
           │
           ▼
  4. EMAILS IN INBOX (HTML formatted with styling)
           │
           ▼
  5. REMINDERS (automatic at D+7, D+14, D+30)
           │
           ▼
  6. MANUAL CLOSURE (data preserved)
